# Rarity System

This repo contains a static, single-file guide that can be published via GitHub Pages.

## View locally

Open `docs/tcg-rarity-guide.html` in a browser.

## Publish online (GitHub Pages)

1. In GitHub, go to **Settings → Pages**
2. Under **Build and deployment**
   - **Source**: `GitHub Actions`
3. Save

After Pages finishes deploying, the site will be available at your repo’s GitHub Pages URL.

The entry point is `docs/index.html`, which redirects to `docs/tcg-rarity-guide.html`.
